# 📅 GesEmpl

## 📌 Description

**GesEmpl** est une plateforme de pilotage pédagogique dédiée à l’optimisation des ressources du **CMC Béni Mellal**.  
Elle permet une gestion dynamique et intelligente des emplois du temps en tenant compte de la complexité des infrastructures modernes de la Cité.

---

## 🎯 Objectif du projet

- Optimiser la gestion des emplois du temps
- Gérer les salles, formateurs, modules et groupes
- Réduire les conflits de planning
- Faciliter la planification pédagogique

---

## ⚙️ Technologies utilisées

- React.js ⚛️
- Redux Toolkit 🧠
- React Router DOM 🌐
- CSS3 🎨
- JavaScript (ES6+)

---

## 📂 Fonctionnalités

- 📅 Gestion des emplois du temps
- 👨‍🏫 Gestion des formateurs
- 🏫 Gestion des salles
- 📚 Gestion des modules
- 👥 Gestion des groupes
- 🌙 Mode sombre (Dark Mode)
- 🔄 Navigation fluide entre pages
